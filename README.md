# Deutsch-Flex Data Dashboard

A comprehensive German language learning dashboard featuring advanced data visualizations, interactive components, and gamification elements.

## Features

### Data Visualizations
- **Learning Flow Matrix**: Interactive scatter plot showing accuracy vs. time-on-task across all resource types
- **Resource Pulse**: Animated progress tracking for Books, Audio Tracks, and Video Lessons
- **Skill Analysis**: Radar chart visualizing proficiency across 6 German language skills
- **Weekly Engagement**: Bar charts tracking engagement and completion rates

### Interactive Components
- **Deep-AI Explanations Lab**: OCR text extraction with syntax and semantic analysis
- **Transkriptionen & Lehrerbücher Hub**: 3D book stack with audio waveform integration
- **Flashcard Spintrix**: Rotating flashcard carousel with heat map indicators
- **Interactive Games**: Gamification hub with progress tracking and achievements
- **Data Forensics Tracker**: Visual learning patterns with achievement badges

### Design
- Glassmorphic UI with translucent panels and frosted glass effects
- Vibrant color palette: emerald green, electric blue, sunny yellow, and deep purple
- Animated 3D elements including floating pretzels and German flags
- Responsive layout with mobile-first design

## Tech Stack

- **React** 18.3.1
- **Tailwind CSS** 4.1.12
- **Recharts** 2.15.2 - Data visualization library
- **Motion** (Framer Motion) 12.23.24 - Animation library
- **Radix UI** - Accessible component primitives
- **Lucide React** - Icon library
- **TypeScript** - Type safety
